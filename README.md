# 💱 Универсальный Конвертер Валют (Vanilla JS)

Современное Single Page Application (SPA) для мгновенной конвертации мировых валют. Вдохновлено калькулятором валют от Google Finance: вводите значение в одно поле, и все остальные пересчитываются моментально. 

🚀 **[Смотреть Demo (GitHub Pages)](https://shohruh1997.github.io/course-convertor/)** *(необходимо включить GitHub Pages в настройках репозитория)*

---

## ✨ Ключевые возможности

*   **Мгновенный мульти-пересчет:** Измените любую валюту, и все остальные обновятся автоматически через базовую валюту (USD) под капотом.
*   **Исторические курсы:** Встроенный календарь позволяет узнать курс на любую дату в прошлом.
*   **Свежие данные с Forex:** Используется бесплатный API (open.er-api.com) для актуальных рыночных курсов.
*   **Умный UX:**
    *   При клике на валюту она мгновенно "взлетает" наверх списка — удобно следить за тем, что конвертируете.
    *   Числа автоматически форматируются для чтения (например: `302 501,01` вместо `302501.01`).
    *   Настройки пользователя (выбранные валюты, тема, порядок полей) навсегда сохраняются в `localStorage` браузера.
*   **Темная/Светлая тема:** Подстраивается под вашу ОС и переключается вручную.

## 🛠 Технологии

Проект создан **без фреймворков и библиотек**, чтобы быть максимально быстрым и легковесным:
*   **HTML5** (Семантическая разметка)
*   **CSS3** (CSS-переменные для тем, Flexbox)
*   **Vanilla JavaScript / ES6+** (ES Modules, async/await, Fetch API)

## 🚀 Как запустить локально

Так как проект не требует сборки (Webpack/Vite) или бэкенда, его можно запустить мгновенно:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Shohruh1997/course-convertor.git
   ```
2. Откройте папку проекта и используйте любой локальный сервер (например, плагин *Live Server* в VS Code или `npx serve`):
   ```bash
   npx serve .
   ```
3. Откройте `http://localhost:3000` в браузере.

*(Простое открытие `index.html` двойным кликом может заблокировать CORS при ES-модулях, поэтому локальный сервер рекомендуется).*

## 📡 API

Приложение использует бесплатные сервисы (не требующие API-ключей):
1. [ExchangeRate-API](https://www.exchangerate-api.com) `(open.er-api.com)` — для свежих и точных курсов на текущий день.
2. [Currency-API от fawazahmed0](https://github.com/fawazahmed0/currency-api) — как fallback (запасной) и для работы с историческими курсами (календарь).
